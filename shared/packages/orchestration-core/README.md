# Maestro Orchestration Core

Multi-agent orchestration framework for the Maestro Platform.

## Features

- Persona-based orchestration
- Team organization and coordination
- Session management
- Parallel execution support
- RAG integration
- Autonomous SDLC engine

## Installation

```bash
pip install maestro-orchestration-core
```

## Usage

```python
from maestro_orchestration_core import PersonaOrchestrator, SessionManager

orchestrator = PersonaOrchestrator()
session = SessionManager()

# Orchestrate multi-agent workflow
result = orchestrator.execute_workflow(workflow_def)
```
